01_NXP i.MX95简介

i.MX 95应用处理器提供先进的图形和视频核心，强大的视觉和机器学习加速，高效的CPU性能加上实时处理和先进的安全，集成的EdgeLock®安全领域，以支持节能的边缘计算。

i.MX 95应用处理器集成了多达6个Arm Cortex®-A55核，内置的Arm Cortex®-M33，支持功能安全，它可以配置为一个安全岛。优化工业、物联网和汽车设备的性能和功率效率。i.MX 95处理器采用了NXP创新的能源弹性架构。

i.MX 95应用处理器提供了一套针对汽车、工业和商业物联网市场的丰富外设。这是EdgeVerse™智能边缘解决方案组合的一部分。i.MX 95系列将提供商业、工业、扩展工业和汽车水平的资格认证，并得到NXP的产品寿命计划的支持。

3.5.1 底板电源

底板电源为直流12V，由DC-005插座（P9）引入。VCC_12V电源经过静电防护器件、自恢复保险丝、防反接二极管后，给核心板供电；后经MP8765降为VDD_5V，这个电源受核心板控制，为底板供电；后经过MP2147降为VCC_3V3，这个电源受核心板控制，为底板供电；通过RT9013降为VCC_1V8，这个电源受核心板控制，为底板供电；同时VCC_12V电源分别经过2个MP2307降为PCIE1_3V3与PCIE2_3V3，这两个电源不受核心板控制，分别为2个PCIE供电。

MP8765的EN引脚由核心板输出的VDD_3V3拉高，从而保证VCC_12V经过MP8765输出VCC_5V，进而输出VCC_3V3、VCC_1V8给底板各电路提供电源。

此部分电路是保证了核心板先上电，底板后上电，以防闩锁效应的发生损坏CPU。

3.5.2 开关机键、复位与休眠按键

底板右下方的K1为开发板的休眠键，命令行休眠，按键可以唤醒。

K2为开发板的复位键，按下后可关闭核心板上所有电源，实现给整板断电复位的功能。

K3为开发板的开关机键，功能类似于手机的开关机键，长按该键，可开机，再长按，可关机。

注意：

BTN_WAKE、SYS_RST_B、ONOFF引脚不用时可以悬空。

3.5.4 调试串口

调试串口采用的是TYPE-C的形式，使用XR21V1414芯片将Cortex-A55、 Cortex-M33、Cortex-M7的三个调试串口转换成一组USB总线，方便使用和同时调试三个内核。

UART1是Cortex-A55 调试串口，2线，3.3V电平；UART2是Cortex-M33调试串口，2线，3.3V 电平；UART3是Cortex-M7调试串口，2线，3.3V 电平。

核心板调试串口信号经过缓冲芯片之后再经过转接芯片，最后来到TYPE-C接口，保证了在底板未上电的时候插入调试串口，都不会对核心板和底板产生漏电问题。

3.5.5 ADC接口

核心板支持原生8路12Bit的ADC接口，采样速率1MS/s开发板通过2.54间距的插针引出，并配有一个滑动变阻器，可以 通过将滑动变阻器的中间引脚连接到ADC引脚来实现ADC采集的功能。其中P11的1/2两个引脚为滑 动变阻器中间引脚，剩余8个引脚均为ADC输入引脚，并在PCB上有对应的丝印标识。

3.5.6 音频

开发板上具有一路HP/MIC接口、一路板载MIC接口和两路SPAKER接口。HP/MIC接口使用四段式耳机座引出，同时可以实现录音和放音；板载MIC可以直接实现录音功能；两路SPAKER接口通过2.54间距白色端子座引出。耳机具有插拔检测的功能， 当耳机插入时，放音将会自动切换到耳机通道上来，SPAKER将不再放音。

音频芯片NAU88C22内部自带的D类功放输出端，可驱动两只8Ω喇叭，最高输出功率为1W，如 果需要外接更大的功放，只能从耳机插座获取信号，不能从喇叭接口获取信号。

注意：

喇叭的功率来自D类功放，不是传统的模拟功放；

每个插座接一个喇叭，切记不可共用喇叭线，也不能把喇叭信号接到地线上；

3.5.7 双千兆以太网

核心板最多支持两路千兆网口，开发板上提供了两路千兆以太网接口，均使用RGMII接口形式与 PHY芯片RTL8211相连，由RJ45插座引出，分别为P37和P38，插座型号为FC-H021LNL，内置隔离变压器，两个网口均支持TSN功能。

开发板提供了JTAG调试接口（P22），方便用户通过仿真设备对i.MX95进行仿真调试。

3.5.9 TYPE-C接口

在底板中将USB2.0连接到 Type-C 接口P32用作烧录功能口，同时将USB3.0接入Type-C接口，此TYPE-C接口支持正反插和主从转换。

当在从模式下，主设备（如电脑）通过电缆插入TYPE-C座时，TYPEC_PWR_EN信号将底板给TYPE-C 接口的供电切断，保证从主设备而来的5V与开发板的5V二者不发生冲突；当在主模式下，从设备通过电缆插入TYPE-C座时，TYPEC_PWR_EN信号将底板供电打开，确保给从设备的正常供电。

3.5.10 TF卡接口

开发板具有一路TF卡接口，使用SD2总线，支持SD card 3.0协议，核心板可以自主对其进行高 低速卡的电平切换和电源管理，设计时请使用核心板专门的供电引脚VDD_SD2_3V3对TF卡进行供电，不建议使用其他电源供电。

3.5.11 RTC接口

开发板具有一路RTC电路，挂在与I2C3总线，搭配使用CR2032纽扣电池，可以实现在开发板不 上电的时候保存时间，并实时走时，开发板上电之后会重新从RTC中读取现有时间，并写入系统当中。

3.5.12 SPI接口

开发板提供了一路SPI接口，通过10P插针P10引出一路SPI3。另外，该接口还预留了电源、地等信号，方便用户的扩展。

3.5.13 MIPI DSI接口

核心板支持MIPI DSI 4通道，最大支持4kp30或38401440p60，开发板提供了一个通用的液晶显示接口， 通过一个0.5mm间距30P的FPC座P16引出，可连接我公司生产的MIPI电容触摸屏。该液晶显示接口采用的是MIPI DSI 4通道的接法，分辨率1280800。

3.5.14 LVDS接口

开发板支持2组4-laneLVDS和1组8-laneLVDS，最大支持1080p60，开发板提供了一个通用的液晶显示接口，通过一个2.0mm间距38P的双排插针座P15引出，可连接我公司生产的LVDS电容触摸屏。

核心板通过LVDS_EN控制LVDS的5V电源供电。

3.5.15 OV13850 MIPI CSI接口

核心板原生支持两个MIPI CSI 4lane通道（与MIPI DSI复用一路），最大2.5Gbps速率，符合V1.2协议规范，可以输入1路4Kp60（单通道）、2路4Kp30、4路1080p60、8路1080p30。开发板使用26PIN 的FPC排座引出MIPI CSI功能，该接口独立电源进行供电,可连接飞凌OV13850摄像头模组。（目前由于原厂为开放ISP功能，所以此功能暂不能使用）

3.5.16 4G接口

开发板搭载了MINI PCIE接口，并具有USB协议线，可以支持4G模组等USB协议的MINI PCIE 接口设备。开发板默认支持和搭载EC20模组，可以进行4G上网。该接口有独立的电源供电，目前为 3.3V3A，可满足大部分的模组需求。

3.5.17 USB2.0 HUB接口

开发板在设计上使用USB2.0 HUB芯片 FE1.1S 对原生USB接口进行了拓展，使用USB2总线，拓展之后分别连接到了三个USB-A 母座与4G上。

3.518 485接口

开发板使用核心板现有的两路串口通过电路引出了两路485接口，再符合485标准协议的基础上， 具有自动收发控制的功能，还增加了防护电路，按照国标标准的EMC实验ESD/EFT/SURGE实验四级等级设计，具有高安全性。通过7PIN端子P48引出两路485接口。

3.5.19 CAN接口

开发板使用原生CAN FD引出了五路CAN FD接口，再符合CAN 2.0B协议规范的基础上，还增加 了防护电路，按照国标标准的EMC实验ESD/EFT/SURGE实验四级等级设计，具有高安全性。分别通过10PIN端子引出三路CAN接口，7PIN端子引出两路CAN接口。

3.5.20 WIFI&BT接口

开发板板载WiFi&BT模组，默认焊接AW-CM358SM，WLAN：IEEE 802.11 a/b/g/n/ac 双频WIFI，高达433.3Mbps收发速率；蓝牙：Bluetooth 5 ，速率高达3Mbps 速率。为增强信号质量，请使用2.4&5GHz双频天线。

3.5.21 PCIE3.0

开发板支持两路PCIE 3.0 X1接口，通过PCIE X1标准转接卡插座P35、P36引出，方便用户接入各种PCIE设备。（由于原厂为开放此接口，所以两个PCIE功能不能使用）

3.5.22 XFI接口

开发板支持一路XFI-10G接口，通过光笼子连接器P39引出。由于10G速度对CPU负载量较大，实际测试发送时需要2个核心分摊流量才可以跑满，接收时需要6个核心分摊流量才能跑满。

3.5.23 USER_LED_KEY

开发板将用户LED和用户按键集成到一起，方便用户测试GPIO的输入与输出。需要注意的是，在 测试GPIO输出功能的时候，不要按下按键，否则会将GPIO强制拉至低电平。

3.5.24 BUZZER蜂鸣器

开发板提供一路蜂鸣器LS1，核心板通过GPIO_IO37控制蜂鸣器。

3.5.25 QSPI FLASH

开发板提供1路QSPI FLASH 接口，选取NOR FLASH芯片W25Q256，容量为128M bit。

3.5.26 FAN接口

开发板提供风扇供电接口，直流12V经过自恢复保险丝、防反接二极管后为风扇供电。
